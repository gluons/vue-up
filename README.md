## Installation

::: warning Compatibility
Vue Up requires [Node.js](https://nodejs.org/) >= 6.11.5.
:::

**Install via [npm](https://www.npmjs.com/):**

```bash
npm install --save-dev @gluons/vue-up
```

---

**Install via [Yarn](https://yarnpkg.com/):**

```bash
yarn add --dev @gluons/vue-up
```
